Why Choose Fact Guardian?

FactGuardian is a Chrome Plugin Application that provides users with insight on instances of hateful speech on Twitter. Once a hateful tweet or comment is detected while the user is browsing Twitter on their Chrome browser, there will be a prompt to the user to verify if the user would like to respond to that tweet/comment. If the user decides to respond, FactGuardian will generate an automatic human-like response using Natural Language Processing algorithms and asks the user if they would like to send that response.

Features

  • Chrome Browser Extension that works on Twitter
  • ChatGPT automated responses for hate speech
  • Respond with human like responses
  • Detects hate speech
  • Allows you to post to your Twitter
